## Run Locally
Clone the project
```bash
git clone https://github.com/xingvoong/ants_vs_bees
```
Go to the project directory
```bash
cd ants_vs_bees
```
Run
```bash
python3 gui.py [-h] [-d DIFFICULTY] [-w] [--food FOOD]
```
